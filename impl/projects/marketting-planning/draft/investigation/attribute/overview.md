# Attribute investigation overview

This package maps the public marketing-planning SSE boundary to the checked-out business workflow at revision `222c3d88c05d0c55735cab3709485d1d633c6335`. It supports target/path extraction, workflow handoff, path execution, assembly, SSE and adapter investigations without assuming every planning gap has the same cause.

The reusable tools stop before expensive planning when possible. They execute current business functions and report their source hashes; they do not encode reference answers. The text trace is the operational entry point. The Mermaid file is a machine-readable topology, not sufficient evidence on its own.
