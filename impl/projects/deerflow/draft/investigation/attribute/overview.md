# Deerflow Attribute investigation overview

This package separates two systems that earlier attribution conflated: the deer-flow Gateway/agent that produces message histories, and verifier's `live.py` that selects one AI message and derives reply, tools, scripts and stage. Its primary deterministic capability replays that verifier boundary on an exact Gateway message array.

The package also indexes the current business NBEV skill contract and a separate clarification-to-planning branch. The current Judge gap selects the branch: a confirmed raw/stored mismatch uses the Gateway extraction flow; repeated clarification after cumulative user input uses the clarification/planning flow; unrelated business gaps continue through the relevant source and runtime tools. Static source and historical traces are navigation material, not current-case cause by themselves. Each text trace map is an operational entry point; Mermaid supplies the same topology for machine reading.
