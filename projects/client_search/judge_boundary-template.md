# Judge Boundary - client_search
限制：有些用户受限于当前上游依赖的数据库中没有相应字段或枚举值的，可能无法完全满足用户意图搜索出正确字段，此时应该按照业务系统能力边界判定是否正确
- 因为上游依赖的es数据库能力不是由我们系统能力强弱决定的，我们只能做到在我们能决定的范围内做到最好，所以评估只评价我们当前系统做的怎么样了，而不评估由于上线限制导致的偏差（指我们无法控制的es数据库这种），这种问题我们无法衡量和解决
- 但如果是我们可以优化的部分（比如大模型、项目配置、项目代码等）做的不好，那就应该纳入衡量范围里面
- 具体数据库相关信息，请从projects/client_search/readme.md、config.md、prompt.md、原项目es信息中进行找出来
